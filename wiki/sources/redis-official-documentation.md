## Что изучено

- Data types and commands: Strings, Hashes, Lists, Sets, Sorted Sets, Streams, HyperLogLog, Bitmaps, Geo.
- Persistence and reliability: RDB, AOF, rewrite, replication, Sentinel, Cluster behavior.
- Performance and operations: memory management, expiration, eviction, observability, latency troubleshooting.
- Concurrency patterns: Lua/Functions, distributed locks, rate limiting, pub/sub and streams trade-offs.
